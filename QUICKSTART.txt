MOVIE TRACKER APP - QUICK START
================================

INSTALLATION (Run Once)
-----------------------
npm install --legacy-peer-deps
npx expo install expo-asset expo-font


SETUP CREDENTIALS (Run Once)
-----------------------------
1. Copy template:
   cp .env.example .env

2. Get Firebase credentials:
   - Go to: https://console.firebase.google.com/
   - Create project
   - Enable Authentication (Email/Password)
   - Create Firestore Database
   - Get config from Project Settings

3. Get OMDB API key:
   - Go to: https://www.omdbapi.com/apikey.aspx
   - Request FREE key
   - Activate via email

4. Edit .env file with your credentials


RUN THE APP
-----------
npm start

Then press 'i' for iOS Simulator (Mac)
   OR press 'a' for Android Emulator
   OR scan QR code with phone (Expo Go app)

⚠️  DO NOT PRESS 'w' - Web mode doesn't work!


IF PROBLEMS
-----------
npm install --legacy-peer-deps
npx expo start --clear


FEATURES
--------
✓ Track movies with IMDB integration
✓ Add ratings and notes
✓ View statistics and charts
✓ Search and filter your collection


TEST IT
-------
1. Sign up with email/password
2. Add movie: https://www.imdb.com/title/tt0111161/
3. Check statistics tab
